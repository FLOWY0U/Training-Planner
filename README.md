# Дневник тренировок

**Автор:** Еникеев Артур

## Описание
Программа на Python с графическим интерфейсом (Tkinter) для ведения лога тренировок.
Позволяет записывать дату, вид активности и время, а также фильтровать историю.
Данные сохраняются локально в формате JSON.

## Функционал
- Валидация даты и положительных чисел.
- Фильтрация по типу тренировки (поиск по подстроке) и по дате (точное совпадение).
- Автоматическое сохранение и загрузка.

## Примеры использования
1. Запустите `main.py`.
2. Введите дату: `04.05.2026`, тип: `Силовая`, длительность: `60`.
3. Нажмите "Добавить тренировку".
4. Чтобы найти все силовые тренировки, введите "силовая" в поле фильтра и нажмите "Применить".
